# RIDLY i18n Guide

Complete guide to internationalization in RIDLY Mobile.

## Overview

RIDLY provides a complete i18n system with:
- Multi-language support
- Interpolation & pluralization
- RTL support
- Number/date formatting
- Lazy loading translations

## Usage

### Basic Translation

```typescript
import { useTranslation } from '@ridly/mobile-core';

function ProductCard({ product }) {
  const { t } = useTranslation('product');

  return (
    <View>
      <Text>{product.name}</Text>
      <Text>{t('labels.price')}: {product.price}</Text>
      <Button>{t('actions.addToCart')}</Button>
    </View>
  );
}
```

### Interpolation

```typescript
const { t } = useTranslation();

// Translation: "Hello, {{name}}!"
t('greeting', { values: { name: 'John' } });
// в†’ "Hello, John!"

// Translation: "{{count}} items in cart"
t('cart.itemCount', { values: { count: 5 } });
// в†’ "5 items in cart"
```

### Pluralization

```typescript
// Translation file:
// reviews: {
//   zero: "No reviews",
//   one: "{{count}} review",
//   other: "{{count}} reviews"
// }

t('product.reviews', { count: 0 });  // в†’ "No reviews"
t('product.reviews', { count: 1 });  // в†’ "1 review"
t('product.reviews', { count: 5 });  // в†’ "5 reviews"
```

### Slavic Pluralization

Ukrainian, Russian, and other Slavic languages have complex plural rules:

```typescript
// Translation file (Ukrainian):
// items: {
//   zero: "РЅРµРјР°С” С‚РѕРІР°СЂС–РІ",
//   one: "{{count}} С‚РѕРІР°СЂ",
//   few: "{{count}} С‚РѕРІР°СЂРё",
//   many: "{{count}} С‚РѕРІР°СЂС–РІ"
// }

// Configuration:
import { slavicPluralRule } from '@ridly/mobile-core';

{
  code: 'uk',
  name: 'Ukrainian',
  nativeName: 'РЈРєСЂР°С—РЅСЃСЊРєР°',
  pluralRule: slavicPluralRule,
}

t('items', { count: 1 });   // в†’ "1 С‚РѕРІР°СЂ"
t('items', { count: 2 });   // в†’ "2 С‚РѕРІР°СЂРё"
t('items', { count: 5 });   // в†’ "5 С‚РѕРІР°СЂС–РІ"
t('items', { count: 21 });  // в†’ "21 С‚РѕРІР°СЂ"
```

### Formatting

```typescript
const { formatNumber, formatCurrency, formatDate, formatRelativeTime } = useTranslation();

// Numbers
formatNumber(1234.56);           // "1,234.56" (en) / "1 234,56" (uk)
formatNumber(0.15, { style: 'percent' }); // "15%"

// Currency
formatCurrency(99.99, 'USD');    // "$99.99" (en) / "99,99 $" (fr)

// Dates
formatDate(new Date());          // "March 16, 2026"
formatDate(date, { dateStyle: 'short' }); // "3/16/26"

// Relative time
formatRelativeTime(new Date(Date.now() - 3600000)); // "1 hour ago"
```

### Language Switching

```typescript
import { useLocale } from '@ridly/mobile-core';

function SettingsScreen() {
  const { locale, locales, setLocale, currentLanguage } = useLocale();

  return (
    <View>
      <Text>Current: {currentLanguage.nativeName}</Text>

      {locales.map(lang => (
        <TouchableOpacity
          key={lang.code}
          onPress={() => setLocale(lang.code)}
        >
          <Text>{lang.flag} {lang.nativeName}</Text>
        </TouchableOpacity>
      ))}
    </View>
  );
}
```

## RTL Support

```typescript
import { useTranslation } from '@ridly/mobile-core';

function App() {
  const { isRTL } = useTranslation();

  return (
    <View style={{ direction: isRTL ? 'rtl' : 'ltr' }}>
      <Content />
    </View>
  );
}
```

## Adding Translations

### Default Translations

RIDLY includes default English translations for common UI elements:

```typescript
// Already included:
t('common.actions.add');        // "Add"
t('common.actions.remove');     // "Remove"
t('common.status.loading');     // "Loading..."
t('cart.title');                // "Shopping Cart"
t('checkout.steps.shipping');   // "Shipping"
// ... and more
```

### Custom Translations

```typescript
import { I18nStore } from '@ridly/mobile-core';

// Add translations at runtime
I18nStore.addTranslations('en', 'custom', {
  welcome: 'Welcome to {{storeName}}!',
  sale: {
    title: 'Big Sale',
    discount: '{{percent}}% OFF',
  },
});

I18nStore.addTranslations('uk', 'custom', {
  welcome: 'Р›Р°СЃРєР°РІРѕ РїСЂРѕСЃРёРјРѕ РґРѕ {{storeName}}!',
  sale: {
    title: 'Р’РµР»РёРєРёР№ СЂРѕР·РїСЂРѕРґР°Р¶',
    discount: 'Р—РЅРёР¶РєР° {{percent}}%',
  },
});

// Use with namespace
const { t } = useTranslation('custom');
t('welcome', { values: { storeName: 'RIDLY' } });
```

### Lazy Loading

```typescript
// ridly.config.ts
i18n: {
  defaultLocale: 'en',
  locales: [...],
  loadTranslations: async (locale, namespace) => {
    // Load from your backend or CDN
    const response = await fetch(`/translations/${locale}/${namespace}.json`);
    return response.json();
  },
}
```

## Hooks Reference

| Hook | Description |
|------|-------------|
| `useTranslation(namespace?)` | Main translation hook |
| `useLocale()` | Locale management |
| `useTranslationLoading(locale?, namespace?)` | Loading state |
| `useI18n()` | Full i18n context |

### useTranslation Return Value

```typescript
const {
  t,              // Translation function
  locale,         // Current locale code
  isRTL,          // Is RTL language
  formatNumber,   // Number formatter
  formatCurrency, // Currency formatter
  formatDate,     // Date formatter
  formatRelativeTime, // Relative time formatter
} = useTranslation();
```

### useLocale Return Value

```typescript
const {
  locale,          // Current locale code
  locales,         // All available locales
  currentLanguage, // Current language config
  setLocale,       // Change locale function
  isRTL,           // Is RTL language
} = useLocale();
```
